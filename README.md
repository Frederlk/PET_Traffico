[Traffico](https://pet-traffico.firebaseapp.com/)
---------------------------------------------------------------------------------------------

A fully responsive Traffic Consultant landing page. All elements of the user interaction with the site are animated and create a pleasant feeling of “high cost” when viewing the site, which also positively affects the attitude of potential customers to the company.

### Stack

*   JavaScript
*   SCSS
*   HTML
*   Swiper
*   Gulp
*   Webpack
